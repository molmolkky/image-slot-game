# スロットゲーム

「令和7年7月7日」にちなんで、スロットゲームを作成しました（Claude Sonnet 4が）。

インターネットから拾ってきた6つの画像でスロットをします。

## 遊び方

ブラウザで動作するスロットゲームで、このリポジトリをクローンし、ターミナルで以下を実行するだけで遊べます！

```zsh
open main.html
```
